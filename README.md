# GMAP

Description
===========

Installs a specific vesion of GMAP, mainly to support deFuse on RHEL 6.5.

Requirements
============

## Platform:

* Centos 6.5


Usage
=====

Include this recipe to install GMAP.

    include_recipe "GMAP::source"
